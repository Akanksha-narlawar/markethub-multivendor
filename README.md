# 🛒 MarketHub - Multi-Vendor E-Commerce Marketplace

MarketHub is a full-stack multi-vendor e-commerce platform where customers can buy products, vendors can sell products, and admins can manage the system.

---

## 🚀 Features

### 👤 Customer
- Browse products
- Add to cart
- Checkout system
- Order history

### 🏪 Vendor
- Add / edit / delete products
- Manage stock
- View orders

### 🛠 Admin
- Approve vendors
- Manage users
- Manage products & orders

---

## 🧱 Tech Stack
- React (Vite)
- Node.js + Express
- MongoDB + Mongoose
- JWT Authentication
